Research Areas

I focus on optimization and safety for transportation, novel meta-heuristics and applications in medical imaging.

Meta-heuristic Algorithms

Development of Slender Loris Optimization Algorithm (SLOA) and application of nature-inspired algorithms for routing and load balancing.

Intelligent Transportation Systems (ITS)

POI-based safe route navigation, dynamic safety scoring with LSTM+Attention, ambulance routing optimization.

Medical Image Processing

Segmentation frameworks combining multimodal MRI, DTI and fMRI for tumor delineation and clinical decision support.

Security & Cloud

Data integrity, client-side deduplication, secure schemes for cloud storage.

Ongoing Projects

  • WOMEN_SLOA: Safety-aware navigation system for night-time travelers using SLOA + LSTM attention for safety scoring.
  • Multimodal segmentation pipeline for tumour delineation using transformer-enhanced backbones.
